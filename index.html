Senior Qt/C++ Engineer · Embedded Linux · Multi-Process Architecture

Qt/Python Developer · Desktop Application · Computer Vision UI

Built the Qt Widgets and PyQt desktop client for ReliVision — Relimetrics' commercial AI-inspection platform, a licensed product running in production on the factory floor. Jun 2021 — Feb 2022 · Bitzer, Denmark

HMT700T User Panel — Embedded HMI for Air Handling Units

Frontend Software Developer · LVGL on RTOS · ARM Cortex-A7

Developed the HMT700T user panel — a 7" capacitive-touch HMI for air handling units, built for an embedded device powered by an ARM Cortex-A7 processor running an RTOS. The interface was implemented with LVGL, a lightweight C graphics library, giving operators a full overview of the installed systems: ventilation temperature, ventilation steps and alarms, heat pumps and pipe status, solar panel status and temperature, and more. The UI adapts to reflect whichever devices are actually installed.

The application centres on a main Swipe View widget holding three pages, each controlling and monitoring a specific part of the system. Day-to-day work included building that Swipe View and its pages, developing new controls from existing designs — custom slider widgets, custom tables, and more — and wiring them to the backend controllers to exchange data and register user input. Since the LVGL widgets are written in C, I created C++ wrappers around them for safer, more convenient use.

Deployed and tested the application on real hardware to verify functionality and reliability, debugged and fixed performance issues, and built a background service that updated the application from a new version supplied over the USB port. Worked in a team of 5–7 backend and frontend developers plus a project manager, using Scrum and Jira for task management.

CMS Touch & Switch UI — Aircraft Cabin Control Panels

Frontend Software Developer · Aviation HMI · D-Bus, MQTT & WebSocket

Developed and maintained the GUI application for embedded cabin-control devices (3.5" to 7" displays) installed in aircraft, giving passengers and crew control over lighting, volume, video sources, window shades, seat adjustments, and more. Built the interfaces in QML on the Qt Framework, with the application backend written in C++ using Qt classes.

Developed the application's communication layer across multiple stages, incorporating D-Bus IPC, a Qt MQTT client in QML and JavaScript, and a WebSocket client in QML for real-time JSON message transmission.
